Big-Data-Processing-Engine

Um sistema de processamento de Big Data de nível empresarial

Overview

This project, Big-Data-Processing-Engine, developed by Gabriel Demetrios Lafis, is an enterprise-grade system designed for efficient processing and analysis of large datasets. It leverages modern Java features and concurrent programming to handle data records, generate insightful summaries, and provide actionable recommendations.

Este projeto, Big-Data-Processing-Engine, desenvolvido por Gabriel Demetrios Lafis, é um sistema de nível empresarial projetado para o processamento e análise eficientes de grandes conjuntos de dados. Ele aproveita os recursos modernos do Java e a programação concorrente para lidar com registros de dados, gerar resumos perspicazes e fornecer recomendações acionáveis.

Features / Funcionalidades

  • Data Ingestion / Ingestão de Dados: Efficiently handles the collection and storage of data records. / Lida eficientemente com a coleta e armazenamento de registros de dados.
  • Concurrent Processing / Processamento Concorrente: Utilizes ExecutorService for parallel data processing, ensuring high throughput. / Utiliza ExecutorService para processamento paralelo de dados, garantindo alta taxa de transferência.
  • Data Analysis / Análise de Dados: Calculates summary statistics (average, min, max, total records) and identifies key insights. / Calcula estatísticas de resumo (média, mínimo, máximo, total de registros) e identifica insights chave.
  • Recommendation Engine / Mecanismo de Recomendação: Generates data-driven recommendations based on analysis. / Gera recomendações baseadas em dados a partir da análise.
  • Modular Design / Design Modular: Built with a clear separation of concerns, making it extensible and maintainable. / Construído com uma clara separação de preocupações, tornando-o extensível e de fácil manutenção.

Getting Started / Primeiros Passos

Prerequisites / Pré-requisitos

  • Java Development Kit (JDK) 17 or higher / Java Development Kit (JDK) 17 ou superior
  • Apache Maven 3.8.6 or higher / Apache Maven 3.8.6 ou superior

Installation / Instalação

git clone https://github.com/GabrielDemetriosLafis/Big-Data-Processing-Engine.git
cd Big-Data-Processing-Engine
mvn clean install

Usage / Uso

java -jar target/big-data-processing-engine-1.0.0-jar-with-dependencies.jar

This will initialize the system with sample data, process it, and print the analysis results to the console. / Isso inicializará o sistema com dados de exemplo, os processará e imprimirá os resultados da análise no console.
